Очень быстрая сортировка
Time Limit: 2000 ms
Memory Limit: 1024 kb

В общем случае для сортировки N объектов нужно сделать не менее log2N! сравнений. Однако знание особенностей сортируемых объектов позволяет произвести сортировку на много быстрее.

Input

На первой строке целое число N (1 <= N <= 100000). Далее следуют N строк, каждая из которых содержит целое число от 1 до 100.

Output

Вывести эти числа в порядке неубывания. Каждое число должно быть на отдельной строке.

Sample Input

5
4
3
2
4
1

Sample Output

1
2
3
4
4
